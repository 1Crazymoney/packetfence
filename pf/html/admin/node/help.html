Node - View
Consult, modify and delete network nodes.
Interface to the pfcmd node commands.
This allows you in particular to register a node by changing its state from unreg to reg.

Node - Add
Add a new network node.
Interface to the pfcmd node add command.
You'll use this command only in some rare cases since the Packetfence system will automatically add all nodes which can be detected on the network.

Node - Lookup
Search for a node by its MAC
Interface to the pfcmd lookup node command.
