Administration

Services

Control which pf services are started / stopped.
Interface to the pfcmd service commands.

Logs

Display of the last 500 lines of the PacketFence log files.

Add User

Create a new user for the admin website.
Interface to the htpasswd command and modifies the <conf_dir>/admin.conf file.
Please note that the new user will have exactely the same privileges you have now (modifying the pf configuration, adding violations to users etc.).

UI Options

Define the default page which is opened when you login. Chose the amount of time that graph images are cached. Setting this to 0 will disable graph caching.
