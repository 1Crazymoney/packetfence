Administration - Configuration
Consult and modify the Packetfence configuration
Interface to the pfcmd config commands.
Please note that this is equivalent to modifying the Packetfence configuration file /usr/local/pf/conf/pf.conf. In particular, the changes are not immediately taken into account. You'll have to restart the affected Packetfence services in order to use the new configuration settings.

Administration - Services
Control which pf services are started / stopped.
Interface to the pfcmd service commands.

Administration - Add User
Create a new user for the admin website.
Interface to the htpasswd command and modifies the /usr/local/pf/conf/admin.conf file.
Please note that the new user will have exactely the same privileges you have now (modifying the pf configuration, adding violations to users etc.).

Administration - UI Options
Choose between three different font sizes for the admin website. Define the default page which is opened when you login. Chose the amount of time that graph images are cached. Setting this to 0 will disable graph caching.
