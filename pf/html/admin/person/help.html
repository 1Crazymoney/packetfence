People - View
Consult, modify and delete your Packetfence users.
Interface to the pfcmd person commands.
Please note that when you delete a Packetfence user, all computers registered to this user will also be deleted.

People - Add
Add a new Packetfence user.
Interface to the pfcmd person add command.
You'll use this command only in some rare cases since a new packetfence user account allows you as an administrator to associate computers with this Packetfence user but it does not allow the user to do so himself/herself. How network users trying to register nodes will be authenticated is determined by the registration.auth configuration instruction and when such an authenticated network user registers his first computer the corresponding Packetfence user account whill be created automatically.
