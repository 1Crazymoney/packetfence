Configuration Reference - vlan

adjustswitchportlvanreasons (multi)
Options: node_modify|manage_register|manage_deregister|manage_vclose|manage_vopen|violation_modify|violation_add|violation_delete
Default: node_modify,manage_register,manage_deregister,manage_vclose,manage_vopen,violation_modify,violation_add|violation_delete
After which calls to pfcmd do we have to re-calculate and re-assign the switchport VLAN a node is connected to

adjustswitchportlvanscript (text)
Default: /usr/local/pf/bin/flip.pl
Which script do we have to execute to adjust the switchport VLAN

closelocationlogonstop (toggle)
Default: disabled
Options: enabled|disabled
Should open locationlog entries be closed when pfsetvlan is stopped.

nbtraphandlerthreads (text)
Default: 20
Number of trap handler threads pfsetvlan should start

nbtrapparserthreads (text)
Default: 5
Number of trap parser threads pfsetvlan should start
