Configuration Reference - expire

iplog (time)
Default: 0
Time which you would like to keep logs on IP/MAC information. A value of 0 disables expiration.
example:
iplog=180d

locationlog (time)
Default: 0
Time which you would like to keep logs on location information. Please note that this table should not become too big since it could degrade pfsetvlan performance. A value of 0 disables expiration.
example:
locationlog=180d

node (time)
Default: 0
Time before a node is removed due to inactivity. A value of 0 disables expiration.
example:
node=90d
