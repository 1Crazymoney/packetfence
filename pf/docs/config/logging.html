Configuration Reference - logging

facility (text)
Default: auth
Syslog facility to log on.

priority (text)
Default: info
Syslog priority to log at.

verbosity (numeric)
Default: 4
Logging verbosity level. Still fairly vague, although 4 is good value for day-to-day operation. For minor troubleshooting, use 8. To see database queries, use 12. For everything, use 20. For only errors, use 1.
