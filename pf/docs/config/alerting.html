admin_netbiosname (text)
Default: EXAMPLE
NetBIOS name of administrative workstation to send alerts with "winpopup" action assigned.

emailaddr (text)
Default: pf@localhost
Email address to which notifications of rogue DHCP servers, violations with an action of "email", or any other PacketFence-related message goes to.

fromaddr (text)
Email address from which notifications of rogue DHCP servers, violations with an action of "email", or any other PacketFence-related message are sent.

log (text)
Default: violation.log
Log file where "log" actions are sent.

smtpserver (text)
Default: localhost
Server through which to send messages to the above emailaddr. The default is localhost - be sure you're running an SMTP host locally if you don't change it!

subjectprefix (text)
Default: PF Alert
Subject prefix for email notifications of rogue DHCP servers, violations with an action of "email", or any other PacketFence-related message.

wins_server (text)
Default: 192.168.0.100
WINS server to resolve NetBIOS name of administrative workstation to IP address.
